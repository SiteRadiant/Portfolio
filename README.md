# Site Radiant

Premium, cinematic one-page Site Radiant website built with React + Vite and GSAP.

## Features
- Cinematic hero with layered motion
- Case studies, packages, methodology, and testimonials
- Custom cursor, subtle noise texture, and premium styling
- Responsive layout across desktop and mobile

## Getting started
- Install: `npm install`
- Run dev server: `npm run dev`
- Build: `npm run build`
- Preview: `npm run preview`

## Notes
- Media panels are placeholders and should be replaced with real assets.
- Copy is aligned to the PRD and can be refined by the content team.
